### 🔹 [Food Finder Recipe Application](https://github.com/itachi200218/Food-Finder-Api-test)  
A full-stack web application for searching and exploring recipes with intelligent suggestions.  

**Tech Stack:**  
- **Backend:** Python (Flask)  
- **Frontend:** HTML, CSS, JavaScript  
- **Database:** MySQL  
- **Caching & Optimization:** Redis  
- **AI Integration:** AI suggestions for unmatched searches  

**Features:**  
- **Smart Search:** Type a keyword (e.g., "chi") → returns **5 relevant recipes** from the database  
- **Categorized Recipes:** Veg, Non-Veg, and other categories for easy browsing  
- **Fuzzy Matching:** Handles typos and close matches intelligently  
- **AI-powered Suggestions:** If no match is found in DB, AI suggests relevant recipes  
- **Real-time search suggestions** for faster user experience  
- Optimized with **Redis caching** for faster response
